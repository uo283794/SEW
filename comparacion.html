¿Que es Asp.Net Core? Historia Versiones Asp.Net Core vs Asp.Net Evaluacion

ASP.Net vs ASP.Net Core

ASP.Net ASP.Net Core
ASP.Net compila para Windows ASP.Net Core compila para Windows, Mac y Linux
ASP.Net tiene un buen rendimiento ASP.Net Core tiene mejor rendimiento que ASP.Net 4X
ASP.Net se ejecuta en .Net Framework ASP.Net Core se ejecuta en .Net Framework y .Net Core
ASP.Net MVC añadió Web.config, Global.asax, Application Start ASP.Net Core no admitía archivos Web.config y Global.asax. Es compatible con appsettings.json
En ASP.Net MVC necesitamos volver a compilar después del cambio de código. En ASP.Net Core a actualización del navegador principal compilará y ejecutará el código sin necesidad de volver a compilarlo.
